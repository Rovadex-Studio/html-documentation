Package

THEME-SLUG.zip

This file located inside the THEME NAME Theme folder, this is the installable WordPress file you'll need to install or update THEME NAME manually.

THEME-SLUG-child-theme.zi

This file located inside the theme package folder, this is a child theme you can use to make changes to THEME NAME's core theme files. In case of theme modifying We strongly recommend to use the Child Theme then. That way, when the parent theme is updated, your modifications are preserved. You don't have to create your own child theme, just use the particular one zipped child theme.
