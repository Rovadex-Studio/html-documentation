Contact form

In order to setup the contact form and start using it you just need to open the php\contact.php file and find the $address variable that define your e-mail address.
Please note, that contact form should be uploaded to the server

Google Map

In order to setup the google maps you need to process with the following steps:

  • Find the folowing string in HTML file and specify your key - <script src="https://maps.googleapis.com/maps/api/js?key=you_googlemap_key"></script>
    To obtain the key you should preliminary refer to the official documentation
  • Для установки марке, нужно открыть файл scripts.js и найти код To define the marker, open the scripts.js file anf find the following code: 
        var marker = new google.maps.Marker({
        position: new google.maps.LatLng(40.6401038, -73.9495823),
        map: map,
        icon: 'img/icon-map.png'
    });
    In the “position” variable define the google map coordinates
  • To get more detailed information please read the official documentation

Package

THEME-SLUG.zip

This file located inside the THEME NAME Theme folder, this is the installable WordPress file you'll need to install or update THEME NAME manually.

THEME-SLUG-child-theme.zi

This file located inside the theme package folder, this is a child theme you can use to make changes to THEME NAME's core theme files. In case of theme modifying We strongly recommend to use the Child Theme then. That way, when the parent theme is updated, your modifications are preserved. You don't have to create your own child theme, just use the particular one zipped child theme.

documentation.html
